Platform Overview

How the fake news detection app is positioned

This project presents news screening in a clearer, more product-like way while keeping the underlying Flask workflow simple.

01

Why It Matters

Misleading or manipulated stories spread quickly, especially when headlines are emotional, vague, or unsupported by reliable sources.

02

How Screening Works

Users submit a short piece of news text. The app sends it for model analysis and returns a result label, confidence value, explanation, and warning indicators.

03

Tech Stack

Flask, Jinja2, SQLite, JavaScript, HTML, CSS, and Gemini API integration power the current workflow.

Operational Flow

Text submission, request validation, model analysis, structured response parsing, result display, and optional account-based storage.
