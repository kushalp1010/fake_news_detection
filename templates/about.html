About This Project

A simple college-friendly Flask project for understanding fake news detection concepts.

What is Fake News?

Fake news is false, misleading, or manipulated information presented as if it were true news. It can spread quickly on social media and mislead readers.

How This Project Works

The user enters a headline or short article. Flask sends that text to the Gemini API with a JSON prompt. The system then shows a result, confidence score, explanation, and warning signs.

Technologies Used

Python 3, Flask, Jinja2, SQLite, HTML5, CSS3, JavaScript, Google Gemini API, and Werkzeug security utilities.

Project Flow

Input Text -> Flask Validation -> Gemini Analysis -> JSON Parsing -> Result Display -> Optional History Save
